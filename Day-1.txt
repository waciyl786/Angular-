1. What is Angular?
         Angular is a JavaScript Framework developed by Google 
		 It provide complete environment to developers 
		 Angular is used to create single page application  and progressive  web apps
		 Framework provide a complete solution developer needs from UI to Testing 
		 Framework also provide structure to the languages to simplify codebases 

2. What is the difference between React and Angular?
       Angular is a framework and React is a library
       Angular is developed by Google and React is developed by meta 
       Angular provide more structure format and React provides more customised format
       Angular provides two way data comunication and React provides one way data communication.
